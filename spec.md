# Specification

## Summary
**Goal:** Build a fully static, single-page marketing landing page for "Vocal Trace," a toddler speech app, with a cohesive brand design system, playful UI motifs, and all specified sections.

**Planned changes:**
- Apply global design system: cream (#FEF9F3) background, navy (#1A365D) headings, teal (#2DADA3) accents, border-radius 32px everywhere, heavy sans-serif font (Nunito or Poppins via Google Fonts)
- Create reusable "Magic Jelly" 3D glossy teal button component with inner glow, drop shadow, and squish-on-press active state
- Build sticky navbar with SVG teal rocket-checkmark logo icon + "Vocal Trace" navy text, center nav links, and a small Magic Jelly "Get Started" button; add scroll shadow
- Build Hero section with exact headline "Watch Their First Words Take Flight.", sub-headline, large "Start the Journey" Magic Jelly CTA, and split-screen layout (toddler illustration left, app mockup with peeking character right)
- Add Social Proof Bar with three grayscale badge icons (Parent's Choice Award, Speech Pathology Approved, STEM Kids 2026) in a horizontal row with dividers
- Build "How It Works" section with three equal columns: Listen / Interact / Grow, each with teal rounded icon container, step number, bold title, and copy
- Build Feature Showcase with two alternating rows (Tactile Feedback / Safe Exploration), each with illustration and copy inside a rounded card
- Build Testimonial section with a large soft teal/navy-tinted rounded card, decorative quotation mark, and the specified Sarah M. quote
- Add Footer CTA section with bold headline and "Unlock Their Voice Today" Magic Jelly button, plus minimal footer with logo, copyright, Privacy Policy and Terms links
- Implement scroll-driven teal dot-trail decorative motif that progressively reveals along the page as the user scrolls

**User-visible outcome:** Visitors see a polished, fully responsive static landing page for Vocal Trace with playful branding, 3D jelly-style buttons, illustrated sections, and a smooth scrolling dot-trail connecting all content sections.
